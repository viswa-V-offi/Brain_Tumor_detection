# Brain_Tumor_detection
AI-powered Brain Tumor Detection System using CNN, TensorFlow, Flask, HTML, and CSS with 96.78% validation accuracy.
# Brain Tumor Detection Using Deep Learning

## Project Overview
This project detects and classifies brain tumors from MRI images using a Convolutional Neural Network (CNN).

## Classes
- Glioma
- Meningioma
- Pituitary
- No Tumor

## Technologies Used
- Python
- TensorFlow
- Flask
- HTML
- CSS

## Dataset
MRI Brain Tumor Dataset

## Accuracy
Validation Accuracy: 96.78%

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run:

python app.py
